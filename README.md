ПО для токового драйвера управления светодиодной панелью на базе микроконтроллера STM32F103C6T6A: реализация протокола консольного управления устройством по интерфейсу USB CDC, алгоритмов измерения и автоматического регулирования выходного напряжения и тока, а также калибровки измерительных каналов.
