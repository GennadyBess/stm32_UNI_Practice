# ПО для токового драйвера светодиодной панели

Прошивка для микроконтроллера **STM32F103C6T6A**, обеспечивающая автоматическое регулирование напряжения/тока, калибровку каналов измерения и консольное управление устройством по интерфейсу USB.

## Шаги для запуска

1. **Клонирование репозитория:**
   
   ```bash
   git clone https://github.com/GennadyBess/stm32_UNI_Practice
2. **Импорт проекта:**
  * Откройте STM32CubeIDE.
  * Выберите _File_ ➔ _Open Projects from file system_.
  * Укажите путь к каталогу с проектом.
3. **Прошивка**
  * Подключите плату к ПК по USB
  * Подключите ST-Link v2 к контактам SWD платы (SWDIO, SWCLK, GND, 3.3V).
  * Нажмите кнопку Run в STM32CubeIDE в файле `main.c`.
4. **Использование:**
  * Переподключите плату к ПК по USB
  * Откройте любой серийный терминал (PuTTY, Tera Term, Serial Studio) на появившемся COM-порту (порт можно посмотреть в диспечере устройств)
  * Введите команду `get` для проверки работоспособности.

## Структура проекта (основные файлы)

* `main.c` — Точка входа в программу, инициализация периферии и главный цикл выполнения.
* `cli.c` — Обработка команд пользователя и логика выполнения калибровочных процедур.
* `utils.c` — Вспомогательные утилиты (работа со строками и числами, установка точности вывода).
* `mcp41010.c` — Драйвер для работы с цифровым потенциометром MCP41010.
* `usbd_cdc_if.c` — Прием байт по USB, накопление их в символьный буфер команд и отправка ответов.

## Оптимизация памяти

* **Текущий размер прошивки:** **30.10 кБ** *(лимит для STM32F103C6T6A: 32 кБ; из-за использования библиотеки [EEPROM Emulation](https://github.com/nimaltd/ee) реальный лимит: 31 кБ)*.
* **Потенциал оптимизации:** Размер прошивки можно уменьшить примерно на **5 кБ**. Для этого в файле `cli.c` внутри функции `process_command` необходимо заменить стандартную библиотечную функцию `strtok` на собственную реализацию разбора строк.
