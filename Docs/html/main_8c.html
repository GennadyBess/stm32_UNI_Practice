◆ current_regulation_pi()

void current_regulation_pi ( float v_feedforward,
float t_target )

Выполнение одного шага PI-регулирования тока.

Рассчитывает необходимую коррекцию напряжения на основе ошибки по току, объединяет её с расчетным напряжением модели (Feedforward) и устанавливает новое целевое напряжение для аппаратного регулятора.

  • Parameters
    [in]v_feedforwardРасчетное напряжение модели (V_out2). Если < 0, берется старое значение.
    [in]t_targetЦелевой ток (А). Если < 0, используется текущий целевой ток.

◆ voltage_regulation_pi()

void voltage_regulation_pi ( int16_t p_feedforward,
float t_target )

Выполнение одного шага PI-регулирования напряжения.

Функция опрашивается в главном цикле по флагу готовности измерений (каждые 50 мс). Реализует формулу Feedforward + PI с защитой интегратора от насыщения (Anti-windup clamping).

  • Parameters
    [in]p_feedforwardРасчетное стартовое положение потенциометра (feedforward от модели). Если передано < 0, используется текущее базовое значение.
    [in]t_targetЦелевое напряжение. Если < 0, используется текущее целевое.
