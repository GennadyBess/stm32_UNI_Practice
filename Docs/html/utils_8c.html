Вычисляет компоненты float числа и добавляет коэффициент калибровки в текстовый буфер.

Функция разбивает вещественное число на целую и дробную части, и записывает получившуюся строку в буфер.

  • Parameters

    [in]pТекущая позиция (индекс) для записи в буфере.
    [in]labelНуль-терминальная строка с именем коэффициента.
    [in]valueВещественное значение коэффициента (float), которое нужно вывести.
  • Returns
    int Новый индекс позиции в буфере после успешной записи всех символов.

◆ custom_atof()

float custom_atof ( const char * str)

Конвертирует строку ASCII в неотрицательное число с плавающей точкой.

Парсит целую и дробную части числа. В качестве разделителя допускается только точка ('.').

  • Note
    Функция не обрабатывает знак минус ('-') и возвращает ошибку при попытке передать первым символом не цифру. Числа больше допустимого будут приведены к диапазону типа данных float.
  • Parameters
    [in]strУказатель на исходную нуль-терминальную строку.
  • Return values
    >=0Успешно декодированное число.
    -1.0fОшибка парсинга (неверный формат или некорректные символы).

◆ custom_atoi()

int32_t custom_atoi ( const char * str)

Конвертирует строку ASCII в неотрицательное целое число (от 0 до 65_535).

Функция посимвольно парсит строку, пока не встретит символ, не являющийся цифрой от '0' до '9'.

  • Note
    Функция не обрабатывает знак минус ('-') и возвращает ошибку при попытке передать первым символом не цифру. Числа больше допустимого будут приведены к диапазону 0-65_535.
  • Parameters
    [in]strУказатель на исходную нуль-терминальную строку.
  • Return values
    >=0Успешно декодированное число.
    -1Ошибка парсинга (строка пуста или первый символ не является цифрой).

◆ custom_utoa()

int custom_utoa ( uint32_t value,
char * buf,
int digits,
char pad_char )

Конвертирует беззнаковое 32-битное число в строку ASCII с форматированием.

Функция переводит число в текстовый вид, выравнивая его по правому краю и заполняя пустые левые разряды указанным символом-заполнителем (например, нулями или пробелами). Строка записывается в буфер БЕЗ завершающего нуля (нуль-терминатора), что позволяет легко склеивать строки.

Parameters
[in]valueПреобразуемое число (uint32_t).
[out]bufУказатель на буфер в памяти, куда будет записан результат.
[in]digitsМинимальное количество символов, которое должно быть записано.
[in]pad_charСимвол для заполнения пустых левых разрядов (например, '0' или ' ').
Warning
Длина Value должна быть НЕ БОЛЬШЕ значения digits. Соответствующей проверки нет.
Returns
int Количество фактически записанных в буфер символов (всегда равно параметру digits).
